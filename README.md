# Global Sales Dashboard (Power BI)

## 📊 Overview
Interactive dashboard built in Power BI to analyze global sales performance.

## 🎯 Objective
Provide insights into sales data, including revenue, categories, and geographic distribution.

## 🛠️ Tools Used
- Power BI

## 📈 Features
- Total sales KPI
- Sales by category and subcategory
- Sales distribution by country
- Order analysis by priority
- Interactive filters (year, segment, country)

## 📚 Learning
Project developed to practice data visualization and business analysis using Power BI.
